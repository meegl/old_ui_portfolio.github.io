Beatslive iPad Remote Monitoring

  • Prev /
  • Next

Beatslive takes an innovative approach to realtime object geolocalization and tracking via GPS. Think of objects connected to the mobile web who send out information to the user. Position is plotted on Google Maps along with additional information.

Think of Beatslive as the Twitter for things.

Mortensen designed the user interface (still in development) that allows for the tracking of objects and people in realtime and gives the user the ability to get notifications via Twitter, email or SMS of any given event.

Additionally we also created the product website for the project.
