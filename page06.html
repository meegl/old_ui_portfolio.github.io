El Gran Dictat TV3 Show iPhone/iPad Game

  • Prev /
  • Next

El Gran Dictat (The Big Dictation) is a TV quiz show where contestants must pass spelling related tests that get more complex as the game goes.

The goal was to translate the TV format first to the web and later to iPhone/iPad/Android

platform. The main challenge was to make the user feel as she was a contestant on the TV show while taking advantage of all the iPhone/iPad interactive capabilities.

As we are all iPhone users, we designed the game we'd like to play.
